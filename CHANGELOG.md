## [0.1.0] - 2024-10-16
### Added
- Basic functionality for daily permit automation.
- Add CLI tool and commands:
  - `gmu c` to configure and get set up.
  - `gmu sc` to show config.
  - `gmu dp` for purchasing daily permits.

### Changed
- Refactored code for better readability.

### Fixed
